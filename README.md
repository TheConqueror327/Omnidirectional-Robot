# Omnidirectional-Robot
## Hardver
### Kerekek (omnidirekcionális / mecanum):
- általunk tervezett & 3D nyomtatott
- rendelt
### Motorvezérlő-modul:
- L298N board
### Motor:
- 12V DC
- ? rpm
- 4 db Mecanum coupling (ha a HESTORE-tól rendelünk előre gyártott kerekeket)

- ESP32 devboard
### Akkumulátor:
- 12V
- ? mAh

### Váz:

### Szerelés:
- breadboard
- kábelek (FF, FM, MM)
